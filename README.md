### Setup
```
git clone git@github.com:ssiyad/exp-pri-ecom.git exp-pri-ecom
cd exp-pri-ecom
npm ci
npx prisma generate
npx prisma migrate deploy
```
### Build
```
npm run build       # or
npm run build:dev   # watch mode
```
### Run
```
npm run start       # or
npm run start:dev   # watch mode
```
